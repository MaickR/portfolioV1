Características Principales
  • Diseño responsive y mobile-first adaptado a todos los dispositivos

Bootstrap 5 SASS Gulp JavaScript PHP

The Clandestino USA - Restaurante

Descripción del Proyecto

The Clandestino es un sitio web completo para un restaurante de vino y tapas españolas ubicado en Mount Shasta, California. Este es uno de mis proyectos más recientes y completos, actualmente en producción a través de Hostinger. El sitio combina jQuery y CSS para crear una interfaz moderna, con PHP y MySQL en el backend para garantizar funcionalidad robusta y gestión eficiente de datos.

Características Principales
  • Sistema de reservas online en tiempo real
  • Menú digital interactivo con imágenes de alta calidad
  • Panel de administración para gestión de menús
  • Galería de fotos del restaurante y platos
  • Integración con Google Maps para ubicación
  • Sistema de notificaciones por email
  • Sección de eventos especiales y promociones
Tecnologías Utilizadas
SASS Bootstrap 5 JavaScript PHP
Resultados

El restaurante experimentó un aumento del 35% en las reservas online desde el lanzamiento del sitio. La tasa de rebote disminuyó un 25% y el tiempo promedio en el sitio aumentó a 5 minutos, lo que indica un mayor engagement de los visitantes.
